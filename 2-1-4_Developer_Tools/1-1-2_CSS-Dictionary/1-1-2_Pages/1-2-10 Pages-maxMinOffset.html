offSet

offset - Is a shorthand, and specifies how to animate an element along a path offset-anchor - Specifies a point on an element that is fixed to the path it is animated along offset-distance - Specifies the position along a path where an animated element is placed offset-path - Specifies the path an element is animated along offset-rotate - Specifies rotation of an element as it is animated along a path

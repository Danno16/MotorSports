Transform

transform - Applies a 2D or 3D transformation to an element transform-origin - Allows you to change the position on transformed elements transform-style - Specifies how nested elements are rendered in 3D space

Transition

transition - A shorthand property for all the transition-* properties transition-delay - Specifies when the transition effect will start transition-duration - Specifies how many seconds or milliseconds a transition effect takes to complete transition-property - Specifies the name of the CSS property the transition effect is for transition-timing-function - Specifies the speed curve of the transition effect

Translate

translate - Specifies the position of an element
